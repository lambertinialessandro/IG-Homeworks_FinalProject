## 🎮 Play the game

<details><summary><b>Show commands</b></summary>

- `W` `A` `S` `D`: directional movement
- `SPACE`: jump
- `SHIFT`: run
- `Mouse`: move the camera
- `ESC`: pause

</details>

<details><summary><b>Show commands (Fly mode)</b></summary>

- `W` `A` `S` `D`: directional movement
- `SPACE`: go up
- `SHIFT`: go down
- `Mouse`: move the camera
- `ESC`: pause

</details>
